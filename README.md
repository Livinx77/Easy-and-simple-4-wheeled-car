# Easy-and-simple-4-wheeled-car
Personal project made by me bc its hard to find an easy vehicle to make with very little resources. I am using lafvin hardware. Its my first time using github. Can be modified and used by anyone.

Did this project because most projects on internet are either hard, need a lot of resources, or they need a 3d printer (which me and a lot and people dont have, and if you want to, the measurements are useful still).
I AM NOT TRYING TO SAY THAT HARDER OR RESOURCEFUL PROJECTS ARE WORSE.
Even if this project was tought to be made with inspiration from a diferent one, i didnt like them, so i decided to make it myself
This project has a very low aim, but im planing to use it as a base for bigger, better and greater projects, i might add things here still. Stay tuned.
Project was also inspired by Dr Stone lol

The car is able to:

Move back and forth (easier with bluetooth module)

Turn left and right (easier with bluetooth module)

Its highly modifiable, it can transform into many things just by changing the chasis, its essentialy a skeleton for anything.

Easy to mod, change and make (20 minutes max of building) 

Flexible measurements
